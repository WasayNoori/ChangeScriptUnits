Sketching geometry features becomes almost second nature when you've worked with SOLIDWORKS even for just a few months.
 
While basic sketch features like lines and circles can be used to create most types of geometry you'll be working with, SOLIDWORKS has several more advanced options available for you to use when creating more complex shapes, such as Polygons, Ellipses, Partial Ellipses, Parabolas, Conic Curves, and Equation Driven Curves. When sketching with basic tools, I'm sure you found that it's much easier to create a rectangle with the rectangle tool than to manually sketch a rectangle with lines and dimensions. The same principle applies when you work with more advanced techniques.
 
Say I want to create a hexagon; I could sketch out the six sides with the line tool, but it would take several constraints or dimensions to control accurately. If I instead use the polygon tool, I can quickly create the multi-sided shape I'm looking for, as well as easily define it with only a single constraint and dimension. I'll go more into Polygons in the next lesson, but for now, know that using these tools can be a huge time saver, and allows for more complexity and visual interest to your designs.
